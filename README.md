# 🏠 张姐的私房菜谱

家庭美食菜单管理微信小程序，让做饭和吃饭都有条不紊。

## ✨ 功能一览

| 模块 | 功能 |
|---|---|
| 🏠 家庭 | 创建/邀请码加入，四角色权限（家长/大厨/干饭人/花朵） |
| 📖 菜品库 | AI 智能推荐、手动添加、批量管理分类/公开、回收站、常点排行 |
| 📋 菜单 | 每日早中晚安排、本周视图、烹饪计时、做饭人显示 |
| 📅 预定 | 多选提交、自动写入菜单+采购清单、他人预定可见 |
| 🛒 采购 | 预定自动汇总、AI 生成、手动增删、批量勾选、分享复制 |
| ⭐ 评分 | 星级评价、实时均分 |
| 👤 个人 | 昵称头像、口味偏好、忌口管理 |
| 🌍 公开 | 菜品可公开分享，跨家庭一键引入 |
| 🤖 AI | DeepSeek 驱动：菜品推荐、采购清单生成 |

## 🛠 技术栈

- **前端**：微信小程序原生
- **后端**：腾讯云开发 CloudBase（云函数 + 文档数据库）
- **AI**：DeepSeek API（chat 模型）
- **认证**：微信 OPENID 自动登录

## 🚀 快速开始

1. 微信开发者工具打开项目
2. 云开发控制台创建 7 个数据库集合（users / families / dishes / menus / preorders / shopping_lists / cook_history）
3. 部署全部云函数（右键 → 上传并部署：云端安装依赖）
4. AI 云函数配置环境变量 `DEEPSEEK_API_KEY`
5. 编译运行

## 📁 项目结构

```
├── cloudfunctions/     # 16 个云函数
├── components/         # dish-card、empty-state
├── pages/              # 10 个页面
│   ├── home/           # 首页（菜单+预定摘要）
│   ├── dishes/         # 菜品库
│   ├── dish-detail/    # 菜品详情
│   ├── dish-add/       # 添加菜品（含AI）
│   ├── preorder/       # 菜品预定
│   ├── preorder-list/  # 预定总览
│   ├── shopping/       # 采购清单
│   ├── profile/        # 个人中心
│   ├── family/         # 家庭管理
│   └── privacy/        # 隐私政策
├── utils/              # 工具函数
└── assets/             # 图标资源
```
