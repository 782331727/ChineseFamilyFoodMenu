# preorder-skill - 预定管理

## 功能说明
提供菜品预定的完整管理能力，包括查看自己的预订单、新增预定、取消预定，以及家长查看和管理全家预定情况。

## 可用原子接口

### getMyPreorders
查看当前用户的所有预订单（跨日期），返回按日期排序的预定列表，包含菜品名称、图片、备注等信息。

### addPreorder
新增菜品预定。需要指定日期、菜品ID、餐次（breakfast/lunch/dinner）和可选备注。同一个用户同一天不能重复预定同一道菜。

### cancelPreorder
取消自己的预定。需要预定的ID，取消前需用户确认。

### getFamilyPreorders
查看全家在指定日期的预定情况（家长可用此了解谁预定了什么、谁还没预定）。
